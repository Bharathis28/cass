## Design Specifications

### Color Codes
```
Neon Purple:    #8A2BE2 / rgba(138, 43, 226, ...)
Cyan Blue:      #1E90FF / rgba(30, 144, 255, ...)
Neon Cyan:      #00E1FF / rgba(0, 225, 255, ...)
Neon Green:     #00ffaa / rgba(0, 255, 170, ...)
Magenta:        #ff00ff / rgba(255, 0, 255, ...)
Dark Purple:    #7f00ff / rgba(127, 0, 255, ...)
White:          #ffffff / rgba(255, 255, 255, ...)
```

### Spacing System
```
Card Padding:        25px
Border Radius:       20px (cards), 12px (headers), 10px (elements)
Section Margin Top:  35px
Section Margin Bot:  20px
Column Gap:          Streamlit default
Min Card Height:     420px (desktop), auto (mobile)
Chart Heights:       450px (Pareto), 350px (Analytics), 320px (Comparison)
```

### Animation Timings
```
fadeLeft:      0.8s ease
fadeInScale:   1.0s ease
fadeRight:     0.8s ease
fadeDown:      0.6s ease
fadeInUp:      0.8s ease
Hover Effects: 0.3s ease
```

### Typography Scale
```
Section Titles:  1.3rem (desktop), 1.1rem (mobile)
Card Titles:     1.2rem
Body Text:       0.95rem
Small Text:      0.8-0.85rem
Large Display:   2.2-2.5rem
```

### Z-Index Layers
```
Floating Badge:  1000
Charts:          1
Cards:           1
Background:      0
```

## Implementation Notes

1. **Glassmorphism Effect:** Achieved through `backdrop-filter: blur(8px)` + semi-transparent backgrounds
2. **Neon Glow:** Applied via `box-shadow` with rgba colors and blur radius
3. **Animations:** CSS keyframes with `animation-fill-mode: both` for persistence
4. **Responsive:** Media query at 768px breakpoint for mobile stacking
5. **Accessibility:** All hover effects maintain sufficient contrast ratios
6. **Performance:** Animations use GPU-accelerated properties (transform, opacity)
